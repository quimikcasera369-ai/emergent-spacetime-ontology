El Lagrangiano del Régimen Colectivo

El modelo predice V⁴ = G·M·g_crit. De esta ecuación se deriva el potencial, y del potencial se construye el Lagrangiano exacto.

Paso 1 — Del potencial a la fuerza

La velocidad de rotación es Rθ̇. Para que la galaxia gire, la fuerza gravitacional debe igualar la aceleración centrípeta. Si V_c = (GM·g_crit)^(1/4) es constante, entonces la fuerza requerida es:

Condición de equilibrio circular
F(R) = m·V_c² / R
Fuerza 1/R — el único tipo de fuerza que produce V = constante para cualquier R.

Paso 2 — El potencial

Integrando F = -dΦ/dR = -mV_c²/R:

Potencial gravitacional colectivo
Φ(R) = V_c² · ln(R) + cte
Potencial logarítmico. Sin radio de referencia fijo — escala libre.

Paso 3 — El Lagrangiano en coordenadas polares

Para una partícula de prueba de masa m moviéndose en el plano (R, θ):

Lagrangiano Colectivo — Definición Formal
L = ½m(Ṙ² + R²θ̇²) − mV_c²·ln(R)
T = ½m(Ṙ² + R²θ̇²) energía cinética. V = mV_c²·ln(R) energía potencial. V_c = (GM·g_crit)^(1/4) constante para una galaxia dada.

Energía total conservada — también Noether exacto.

La Curva Plana — Derivación Directa

No se asume V = constante. Se demuestra que es consecuencia inevitable de la ecuación radial para órbitas circulares.

Condición de órbita circular

Para órbita circular perfecta: R = R₀ = constante. Por lo tanto Ṙ = 0 y R̈ = 0. Sustituyendo en la ecuación radial:

1
Ecuación radial con Ṙ = R̈ = 0
0 = mR₀θ̇² − mV_c²/R₀
2
Despejando
R₀θ̇² = V_c²/R₀  →  (R₀θ̇)² = V_c²
3
Resultado
V_rot = R₀θ̇ = V_c = constante

La velocidad de rotación orbital es V_c = (GM·g_crit)^(1/4) para cualquier radio R₀. La curva de rotación es plana para todo R. Esto no depende de Noether — es consecuencia directa de la ecuación de movimiento.

Derivación exacta completada
V⁴ = GM·g_crit  →  Φ = V_c²·ln(R)  →  F = mV_c²/R  →  V_rot = V_c = const ✓
La curva plana es rigurosa. Sin Noether, sin virial, sin supuestos adicionales.

La Simetría de Escala — Formal

¿Qué pasa exactamente cuando transformamos R → λR? El Lagrangiano cambia, pero las fuerzas no. Aquí está la distinción crucial.

La transformación

Consideremos el reescalamiento simultáneo de espacio y tiempo:

Transformación de dilatación
R → (1+ε)R  ≡  R̃ = e^ε · R
t → (1+ε)t  ≡  t̃ = e^ε · t
θ → θ  (invariante)

¿Qué le pasa al Lagrangiano?

1
Velocidades bajo la transformación
Ṙ̃ = dR̃/dt̃ = (1+ε)dR / (1+ε)dt = dR/dt = Ṙ  ✓
R̃²θ̃̇² = (1+ε)²R²·[θ̇/(1+ε)]² = R²θ̇²  ✓
La energía cinética es exactamente invariante.
2
El potencial bajo la transformación
Φ(R̃) = V_c²·ln((1+ε)R) = V_c²·ln(R) + V_c²·ε
El potencial cambia — pero solo por una constante. Las fuerzas (derivadas del potencial) son invariantes.
3
Variación del Lagrangiano
L̃ = L − mV_c²·ε
4
Variación del elemento de acción L·dt
El punto clave
δ(L dt) = L̃·dt̃ − L·dt
= (L − mV_c²ε)(1+ε)dt − L·dt
≈ ε·(L − mV_c²) dt
Para que Noether aplique, ε(L−mV_c²)dt debe ser una diferencial total dΛ. Esto no se cumple en general fuera de las soluciones de las EOM. Aquí está la caveat.

Lo que SÍ es exactamente simétrico

La fuerza es escala-invariante
F(R) = −dΦ/dR = −mV_c²/R

F(λR) = −mV_c²/(λR) = (1/λ)·F(R)

La fuerza escala como 1/λ — exactamente como la escala espacial.
La ecuación de movimiento mR̈ = F(R) escala homogéneamente bajo R→λR, t→λt. Las trayectorias son geométricamente similares.
Distinción crucial — honestidad matemática
El potencial NO es escala-invariante (cambia por una constante).
La fuerza SÍ es escala-invariante (es de la forma F ∝ 1/R).
Las ecuaciones de movimiento son escala-covariantes (se transforman homogéneamente).

La simetría de escala actúa a nivel de la dinámica, no del potencial.

Noether — El Cargo Conservado

Aplicando el teorema de Noether con la transformación de dilatación, derivamos la carga conservada exacta y verificamos cuándo se conserva.

Fórmula general de Noether

Para transformaciones δq = ε·ξ(q,t) y δt = ε·τ(t), si δL = ε·dF/dt, la carga conservada es:

Carga de Noether — Fórmula General
Q = (∂L/∂q̇ᵢ)·ξᵢ − H·τ + F

donde H = (∂L/∂q̇ᵢ)·q̇ᵢ − L es el Hamiltoniano

Aplicando a la dilatación R → (1+ε)R, t → (1+ε)t

Los generadores de la transformación: ξ_R = R, ξ_θ = 0, τ = t

La función F tal que δL = ε·dF/dt: como δL = −mV_c²ε = cte, tenemos F = −mV_c²·t.

1
Momentos conjugados
p_R = ∂L/∂Ṙ = mṘ
p_θ = ∂L/∂θ̇ = mR²θ̇ = ℓ
2
Hamiltoniano
H = mṘ² + mR²θ̇² − L = ½m(Ṙ² + R²θ̇²) + mV_c²·ln(R) = T + V
3
Carga de Noether para la dilatación
Carga de Noether — Dilatación
Q = mṘ·R + (mV_c² − H)·t

Verificación: ¿dQ/dt = 0?

4
Derivando Q respecto al tiempo
dQ/dt = m(R̈·R + Ṙ²) + (mV_c² − H)

usando E-L: mR̈·R = mR²θ̇² − mV_c²

m(R̈·R + Ṙ²) = mR²θ̇² − mV_c² + mṘ² = 2T − mV_c²

dQ/dt = 2T − mV_c² + mV_c² − H = 2T − H = 2T − T − V

dQ/dt = T − V
Resultado exacto — sin adorno
Q se conserva si y solo si T = V, es decir si T = mV_c²·ln(R₀).

En una órbita circular de radio R₀: T = mV_c²/2 y V = mV_c²·ln(R₀).
T = V  ⟺  ln(R₀) = ½  ⟺  R₀ = √e · R_ref

Esto solo se cumple para un radio específico. Q no es universalmente conservado para la dilatación.

Pero los dos Noether exactos siguen en pie

Simetría rotacional (θ → θ+const) → ℓ = mR²θ̇ = constante
NOETHER EXACTO
Traslación temporal (t → t+const) → H = E = constante
NOETHER EXACTO
Dilatación espacial (R → λR, t → λt) → carga Q, dQ/dt = T−V
QUASI-SIMETRÍA
T = mV_c²/2 = constante en curvas planas
DERIVACIÓN DIRECTA

El Teorema del Virial — La Ruta Limpia

El teorema del virial llega al mismo resultado que la derivación directa, por una ruta completamente independiente — sin Lagrangiano, sin Noether.

Enunciado del virial para fuerzas centrales

Teorema del Virial — Potencial V ∝ Rⁿ
⟨2T⟩ = n·⟨V⟩  (promedio temporal sobre trayectorias acotadas)
Para n=2 (potencial armónico): 2T = 2V → T = V. Para n=−1 (Kepler): 2T = −V. Para n→0 (logarítmico): distinto — ver abajo.

Caso logarítmico — el límite n→0

Para V = mV_c²·ln(R), la forma correcta del virial es:

Virial para potencial logarítmico
⟨2T⟩ = ⟨R · dV/dR⟩ = ⟨R · mV_c²/R⟩ = mV_c²

⟨T⟩ = mV_c²/2 = constante

En órbitas circulares (donde T es exactamente constante, no solo en promedio):
T = mV_c²/2  — exacto, no promediado

¿Por qué T es exactamente constante (no solo en promedio)?

Demostración directa
En órbita circular: R = R₀, Ṙ = 0, V_rot = R₀θ̇

De la condición de equilibrio: V_rot = V_c = (GM·g_crit)^(1/4)

T = ½m·V_rot² = ½m·V_c²  — independiente de R₀ ✓

Como V_c no depende de R, T es la misma en cualquier órbita circular.
Resultado clave
La energía cinética T = mV_c²/2 es constante en todo el plateau colectivo. No porque haya una simetría formal de Noether para la dilatación — sino porque la fuerza 1/R implica una velocidad orbital única para todas las órbitas circulares. El mismo T para todos los radios — eso es la curva plana.
